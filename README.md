# Sensor Calibration Pipeline

Tool for extrinsic calibration between camera and LiDAR sensors, computing the rigid transformation to align 3D point clouds with 2D images.

## Status
🚧 In active development

## Planned Features
- Target-based extrinsic calibration (checkerboard)
- Point cloud to image projection and overlay visualization
- Reprojection error analysis

## Tech Stack
- Python / C++
- OpenCV
- ROS2
